## 🏗️ **Technical Architecture**

### **Backend API Endpoints:**
- `/api/agents/agents` - List all AI agents
- `/api/agents/{agent_id}` - Agent details and controls
- `/api/agents/{agent_id}/start` - Start agent
- `/api/agents/{agent_id}/stop` - Stop agent
- `/api/agents/{agent_id}/configure` - Configure agent
- `/api/agents/activity` - Agent activity logs
- `/api/agents/metrics` - Performance metrics
- `/api/agents/grow-room/status` - Grow room monitoring
- `/api/agents/grow-room/{room_id}/controls` - Environmental controls
